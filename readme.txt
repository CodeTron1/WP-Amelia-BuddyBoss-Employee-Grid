Professional team showcase plugin with Amelia & BuddyBoss integration.

== Description ==
AB Employee Grid and Carousel allows you to display Amelia employees in a responsive grid or carousel with full BuddyBoss integration. Perfect for professional team showcases.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the settings under **AB Employee Grid** to configure.
